PNASC Mobile

Application mobile citoyenne de la Plateforme Nationale d’Alerte et de Signalement Citoyen (PNASC).

Objectif
Permettre aux citoyens de :

Signaler un incident avec photo et géolocalisation

Consulter les incidents proches sur une carte

Voir les alertes officielles publiées par les institutions

Suivre l’évolution de leurs signalements
